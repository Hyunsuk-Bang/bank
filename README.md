# Simple Bank

## Description
Simple Bank Project to practice skills

## Getting Started

### Dependencies
* migrate-golang - tools for DB migration
* sqlc - CRUD code generator for golang

### Executing program
* spawn postgres
    make postgres

* create DB
    make create_db

* drop DB
    make drop_db

* migrate up DB
    make migrate_up
    
* migrate down DB
    make migrate_down
